### 1. 이 블로그 (VitePress Blog)
**기간**: 2024

**기술 스택**: VitePress, Vue.js, TypeScript

**설명**:  
VitePress를 활용하여 만든 정적 사이트 생성 블로그입니다.  
마크다운 기반의 게시글 작성과 자동 빌드/배포를 구현했습니다.

**주요 기능**:
- 마크다운 기반 게시글 작성
- 태그 기반 게시글 분류
- GitHub Actions를 통한 자동 배포
- 다크 모드 지원

**링크**: [GitHub Repository](https://github.com/mslee98/ms-fundamentals)
